# GSA — Visão Arquitetural

Esta pasta contém apenas a **visão arquitetural resumida** do módulo GSA dentro da arquitetura do SimulDIESEL.

## Escopo

O GSA é uma Baby Board especializada, integrada ao Gateway por barramento interno.

Nesta pasta devem permanecer apenas informações de alto nível, como:

- papel do GSA no sistema
- relação com o Gateway
- responsabilidade funcional
- referência para a documentação técnica detalhada

## Fonte de verdade técnica

A documentação detalhada do GSA deve ficar em:

- `05_hardware/gerador-sinais-analogicos-GSA/`

Isso inclui:

- firmware
- protocolo embarcado
- configuração
- pinout
- fluxo de execução
- esquemático, Gerbers e demais artefatos de hardware
