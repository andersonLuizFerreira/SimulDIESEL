# Baby Boards

Esta pasta documenta **a visão arquitetural de integração** dos módulos baby board conectados ao Gateway.

## Escopo desta pasta

Aqui devem ficar apenas documentos de arquitetura de alto nível, como:

- papel do módulo no sistema
- fronteira com o Gateway
- responsabilidade funcional
- posição na arquitetura geral

## Regra de organização

A documentação de **firmware, esquemático, hardware, pinagem, Gerber e detalhes de implementação embarcada**
deve ficar em:

- `05_hardware/`

## Módulos

- `gsa/` — visão arquitetural resumida do Gerador de Sinais Analógicos
