# Hardware

Esta pasta concentra a documentação de hardware e firmware dos módulos do SimulDIESEL.

## Conteúdo

- `gerador-sinais-analogicos-GSA/`

## Regra de organização

Toda documentação técnica detalhada de módulos embarcados deve ficar aqui, incluindo:

- firmware
- protocolo embarcado
- configuração
- pinout
- fluxo de execução
- esquemático
- arquivos Gerber
- demais artefatos físicos e elétricos

## Objetivo

Organizar a documentação física, elétrica e de firmware dos módulos embarcados do projeto.
