# Gateway

Esta pasta documenta a arquitetura lógica do Gateway do SimulDIESEL.

## Conteúdo

- `DEVICE_TABLE.md` — mapeamento de endereços lógicos para barramentos/dispositivos
- `ROUTER.md` — roteamento a partir do campo `CMD`
- `I2C_TLV_CRC.md` — regra de barramento TLV + CRC no enlace com baby boards

## Objetivo

Descrever como o Gateway interpreta comandos do PC e os encaminha para os módulos internos.
