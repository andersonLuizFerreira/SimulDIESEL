# SGGW --- Vetores de Teste

Este diretório contém exemplos de frames SGGW em formato hexadecimal.

Convenções:

-   bytes separados por espaço
-   CRC calculado antes do COBS
-   sequência exata do stream

Arquivos típicos:

-   ping.hex
-   ack.hex
-   event.hex

Utilizados para validar implementações do protocolo.
