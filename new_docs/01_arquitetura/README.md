# Arquitetura — SimulDIESEL

Esta pasta concentra a documentação arquitetural do projeto.

## Subpastas

- `00_contratos/` — contratos centrais entre Gateway e Baby Boards
- `01_protocolos/` — protocolos de comunicação, incluindo SGGW

- `05_babyboards/` — arquitetura dos módulos baby board

## Objetivo

Documentar a estrutura técnica do sistema, suas fronteiras, contratos e fluxos principais.
