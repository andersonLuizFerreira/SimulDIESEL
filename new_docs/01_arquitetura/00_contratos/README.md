# Contratos Arquiteturais

Esta pasta contém os contratos oficiais que definem responsabilidades e formatos de integração entre Gateway e Baby Boards.

## Conteúdo

- `CONTRATO_CENTRAL.md`
- `CONTRATO_GATEWAY.md`

## Objetivo

Estabelecer a fonte de verdade para:

- endereçamento lógico
- roteamento

- compatibilidade entre componentes
