# J1939

## Estado atual

Status: `PARCIALMENTE IMPLEMENTADO`.

A camada J1939 existe no host C# como decodificação acima de `CanFrameDto`. Ela não altera `SDGW`, não pertence ao `SDCTP` e não executa no firmware. O fluxo conservador é:

```text
CAN RX consolidado
  -> CanFrameDto
  -> BLL/Protocols/J1939
  -> FormsLogic/UI
```

Componentes implementados no host:

- parser de CAN ID, PGN, Source Address e Destination Address;
- Data Link J1939;
- Transport Protocol J1939;
- decodificação parcial de aplicação J1939-71 por catálogo local;
- diagnósticos DM1/DM2;
- gerenciamento de rede J1939/81, incluindo Address Claim e registry em memória;
- catálogos J1939/81 locais para identificação de rede CAN;
- janela `FrmRedeCan` somente leitura para snapshots enriquecidos do registry;
- captura temporal J1939 para engenharia reversa, com exportação `.txt`/`.md`.

## Captura temporal J1939

Status: `PARCIALMENTE IMPLEMENTADO`.

A captura temporal fica em:

- `BLL/Protocols/J1939/Capture/J1939TemporalCaptureService.cs`;
- `BLL/Protocols/J1939/Capture/J1939CaptureExportService.cs`;
- `DTL/Protocols/J1939/Capture/`.

Ela é integrada inicialmente na aba `Dados J1939` da tela UCE. A UI inicia e finaliza a captura por `FrmUceLogic`; o serviço recebe apenas mensagens J1939 já produzidas pela pipeline atual da aba, sem reprocessar CAN bruto e sem recriar decoder.

O serviço reduz spam temporal:

- registra a primeira ocorrência de uma mensagem;
- agrupa frames idênticos consecutivos;
- emite eventos `PERIODIC_TICK` com contagem e intervalo médio;
- registra transições por mudança de origem, destino, PGN ou payload;
- enriquece `Address Claim` (`PGN 0x00EE00`) com `ADDRESS_CLAIM`, `NAME`, `ClaimedSA` e `RawCanId`;
- exporta sessão em UTF-8 legível para análise humana e por IA.

O exportador inclui:

- metadados de sessão;
- eventos temporais;
- resumo por PGN;
- resumo por Source Address;
- resumo de Address Claims detectados;
- periodicidade detectada;
- top talkers.

## Limites

- O suporte J1939 é host-side e `PARCIALMENTE IMPLEMENTADO`.
- A captura temporal não é perda zero de frames; ela é uma visão reduzida para engenharia reversa temporal.
- A validação desta ETAPA confirmou build e smoke test de redução/exportação, mas não confirmou captura real em bancada com a EST.
- SDGW, SDCTP, firmware, parser J1939 e fluxo CAN RX consolidado não foram alterados nesta ETAPA.
- Catalogos J1939/81 permanecem parciais e versionados localmente.

## Glossário

- **J1939**: protocolo automotivo diesel usado sobre CAN.
- **PGN**: Parameter Group Number.
- **SPN**: Suspect Parameter Number.
- **SA**: Source Address.
- **Destination Address**: destino J1939, incluindo `GLOBAL` quando aplicável.
- **SDCTP**: SimulDIESEL CAN Transport Protocol, camada de massa/sincronização CAN.
