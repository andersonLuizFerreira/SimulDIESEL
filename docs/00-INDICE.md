# 00-INDICE — Mapa da árvore documental

Este documento apresenta o **tronco principal da navegação oficial**.

Seu objetivo é orientar o leitor sobre os caminhos disponíveis na documentação, permitindo escolher a trilha de aprofundamento mais adequada ao seu objetivo.

A leitura recomendada é sempre **progressiva**, partindo da visão geral até as camadas técnicas.

---

## Como utilizar este mapa

Escolha o ramo de acordo com o que deseja entender no projeto.

### Para entender o projeto como um todo

Siga a trilha conceitual principal:

```text id="3d7quv"
Introdução
→ Objetivos
→ Escopo
→ Arquitetura
→ Camadas
→ Fluxo
→ Hardware / Firmware / Software
```

* [Introdução](official/01-visao-geral/01-introducao.md)
